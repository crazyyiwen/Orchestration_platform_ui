Core design requirement:
Everything must be dynamic and registry-driven.

Create a central node registry like:

{
"llm": {
"type": "llm",
"label": "LLM",
"icon": "sparkles",
"color": "purple",
"defaultName": "llm_0",
"description": "Call an LLM model",
"handles": {
"inputs": [{ "id": "in", "label": "Input" }],
"outputs": [{ "id": "out", "label": "Output" }]
},
"formSections": [...]
}
}

Every node type should declare:

- type
- display label
- icon
- color
- default data
- allowed input handles
- allowed output handles
- dynamic form schema
- output variable schema
- validation rules

Do not create separate React forms like LlmForm.tsx, AgentForm.tsx, RuleForm.tsx unless they are thin wrappers around generic schema rendering. Prefer a generic DynamicFormRenderer.
